***17: Transitions in CSS

Transitions

Transitions enable you to define the transition between two states of an element.


  • transition-property: property you want to transition (font-size,width,all etc.).
  • transition-duration: 2s/4ms.
  • transition-timing-function: ease-in/ease-out/linear/steps(5).
  • transition-delay: 2s/4ms.

  • transition:property name | duration | timing-function | delay;

    • Example:

  • transition:font-size 2s steps(4) 100ms;
